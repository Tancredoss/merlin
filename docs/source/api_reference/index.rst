API Reference
=============

Packages
--------

.. list-table::
   :widths: 30 70
   :header-rows: 0

   * - :doc:`api/merlin.algorithms`
     - Quantum layers, kernels, feed-forward blocks, and training utilities.
   * - :doc:`api/merlin.builder`
     - Circuit construction utilities and builder abstractions.
   * - :doc:`api/merlin.bridge`
     - Conversions between qubit-style and photonic/Fock representations.
   * - :doc:`api/merlin.core`
     - MerLin's main data, processor, components and computation spaces objects.
   * - :doc:`api/merlin.datasets`
     - Dataset loaders and metadata helpers.
   * - :doc:`api/merlin.measurement`
     - Measurement strategies, detectors, mappers, and sampling tools.
   * - :doc:`api/merlin.pcvl_pytorch`
     - Perceval-to-PyTorch conversion and SLOS compute graph utilities.
   * - :doc:`api/merlin.utils`
     - Shared utility helpers for grouping, dtypes, normalization, and deprecations.
